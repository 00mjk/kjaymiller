UPDATE: "Add to Sorted" Now Accepts Lists

- 09/10/2018 22:42

Adding a task to a list is now possible with the Add to Sorted Drafts action.

To add a list, just include a "#" with the list name IMMEDIATELY following it (No Space)

If the list name contains one or more spaces just surround the name with quotations (#"Like This").

The action can be added from the Drafts action directory. You can add it as a stand-alone action or a part of my Sorted 3 Action Group

If you have already downloaded it, you will need to do so again to receive the updated action.

The Drafts and Sorted apps are both available in the iOS App Store.
